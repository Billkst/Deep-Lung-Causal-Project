### 2. Proxy ITE 计算方法修正（已实施）

**原始方法（错误）：**
```
Proxy ITE ≈ P(Y=1|X) - Mean(P(Y=1|X) | T=0)
```
- 问题：存在严重的混杂偏差（Confounding Bias）

**修正方法（反事实干预法）：**
```
Proxy ITE = P(Y=1|X, PM2.5=+1.0) - P(Y=1|X, PM2.5=-1.0)
```
- 原理：通过控制变量法（Ceteris Paribus）剥离基因背景干扰
- 优势：纯粹提取模型眼中的"PM2.5 效应"

## 技术细节

### A. 反事实干预法的数学原理

**定义：**
对于样本 $x = (x_{\text{genes}}, x_{\text{PM2.5}})$，构造两个反事实样本：
- 高暴露样本：$x_{\text{high}} = (x_{\text{genes}}, +1.0)$
- 低暴露样本：$x_{\text{low}} = (x_{\text{genes}}, -1.0)$

**Proxy ITE 计算：**
$$
\text{Proxy ITE}(x) = P(Y=1 | x_{\text{high}}) - P(Y=1 | x_{\text{low}})
$$

**Proxy ΔCATE 计算：**
$$
\text{Proxy } \Delta\text{CATE} = \mathbb{E}[\text{Proxy ITE} | \text{EGFR}=1] - \mathbb{E}[\text{Proxy ITE} | \text{EGFR}=0]
$$

**假设：**
1. 模型的预测概率 $P(Y=1|X)$ 近似于真实概率
2. 模型能够捕捉 PM2.5 与基因的交互效应
3. 反事实干预能够模拟真实的因果效应

**局限性：**
- Proxy ITE 不是真实 ITE，只是一个近似估计
- 依赖于模型的校准性（Calibration）和表达能力
- 不能用于精确的因果推断，仅用于机制验证

### B. 统计功效分析

**检测效应量：** Cohen's d = 0.5（中等效应）  
**显著性水平：** α = 0.05  
**统计功效：** 1-β = 0.80  
**所需样本量（每组）：** n ≥ 64

**实际样本量：**
- EGFR 突变组：n=67 ✅
- EGFR 野生型组：n=446 ✅
- 满足统计功效要求

### C. 数据生成机制回顾

**交互效应公式：**
```
L = -3.0 + 0.086*PM2.5* + 0.69*(PM2.5* × EGFR) + 0.5*sum(Top20_Genes)
P(Y=1) = 1 / (1 + exp(-L))
```

**关键参数：**
- W_INT = 0.69：EGFR 交互效应权重
- W_BASE = 0.086：PM2.5 主效应权重
- W_GENE = 0.5：基因主效应权重

**理论 ΔCATE：**
- 理论值：ΔCATE ≈ 0.69 × σ(PM2.5*) ≈ 0.69 × 1.0 = 0.69
- 实际值（TabR）：Proxy ΔCATE = 0.0633
- 差距原因：Sigmoid 函数压缩 + 模型学习能力限制
