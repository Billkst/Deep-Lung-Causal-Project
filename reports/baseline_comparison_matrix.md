### Proxy ITE 计算方法（反事实干预法）

#### 定义

对于每个样本 $x = (x_{\text{genes}}, x_{\text{PM2.5}})$，构造两个反事实样本：
- **高暴露样本：** $x_{\text{high}} = (x_{\text{genes}}, \text{PM2.5}=+1.0)$
- **低暴露样本：** $x_{\text{low}} = (x_{\text{genes}}, \text{PM2.5}=-1.0)$

#### 计算公式

$$
\text{Proxy ITE}(x) = P(Y=1 | x_{\text{high}}) - P(Y=1 | x_{\text{low}})
$$

$$
\text{Proxy } \Delta\text{CATE} = \mathbb{E}[\text{Proxy ITE} | \text{EGFR}=1] - \mathbb{E}[\text{Proxy ITE} | \text{EGFR}=0]
$$

#### 核心原理

**控制变量法（Ceteris Paribus）：**
- 保持所有基因特征不变
- 仅改变 PM2.5 暴露水平
- 剥离基因背景的干扰
- 纯粹提取模型眼中的"PM2.5 效应"

#### 假设

1. 分类模型的预测概率 $P(Y=1|X)$ 近似于真实概率
2. 模型能够捕捉 PM2.5 与基因的交互效应
3. 反事实干预能够模拟真实的因果效应

#### 局限性

- Proxy ITE 不是真实 ITE，只是一个近似估计
- 依赖于模型的校准性（Calibration）和表达能力
- 不能用于精确的因果推断，仅用于机制验证
- 假设模型已经学到了正确的因果关系

## 数据生成机制验证（白盒审计）

### 交互效应公式

```
L = -3.0 + 0.086*PM2.5* + 0.69*(PM2.5* × EGFR) + 0.5*sum(Top20_Genes)
P(Y=1) = 1 / (1 + exp(-L))
```

### 关键参数

| 参数 | 值 | 说明 |
|------|-----|------|
| W_INT | 0.69 | EGFR 交互效应权重 |
| W_BASE | 0.086 | PM2.5 主效应权重 |
| W_GENE | 0.5 | 基因主效应权重 |
| INTERCEPT | -3.0 | 截距项 |

### 阳性对照与阴性对照

**阳性对照（EGFR）：**
- 仅 EGFR 与 Virtual_PM2.5 存在交互项（W_INT * PM2.5* × EGFR）
- 预期：ΔCATE > 0

**阴性对照（TP53）：**
- 其他 Top20 基因（TP53, KRAS 等）仅参与线性累加（W_GENE * sum(genes)）
- 预期：ΔCATE ≈ 0

### 理论 ΔCATE

- 理论值：ΔCATE ≈ 0.69 × σ(PM2.5*) ≈ 0.69 × 1.0 = 0.69
- 实际值（TabR）：Proxy ΔCATE = 0.0633
- 差距原因：Sigmoid 函数压缩 + 模型学习能力限制
