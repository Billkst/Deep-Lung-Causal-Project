## 修正内容

### 1. Proxy ITE 计算逻辑修正

**原始方法（错误）：**
```python
# 简化估计（存在混杂偏差）
proxy_ite = y_proba - mean_proba_t0
```

**问题：** 存在严重的混杂偏差（Confounding Bias），没有控制住其他基因特征的影响。

**修正方法（正确）：**
```python
# 反事实干预法（控制变量法）
def compute_proxy_ite(model, X, pm25_feature_idx):
    # 构造高暴露样本（PM2.5 = +1.0）
    X_high = X.copy()
    X_high[:, pm25_feature_idx] = 1.0
    
    # 构造低暴露样本（PM2.5 = -1.0）
    X_low = X.copy()
    X_low[:, pm25_feature_idx] = -1.0
    
    # 计算 Proxy ITE
    proxy_ite = model.predict_proba(X_high) - model.predict_proba(X_low)
    
    return proxy_ite
```

**原理：**
- 通过控制变量法（Ceteris Paribus）剥离基因背景干扰
- 纯粹提取模型眼中的"PM2.5 效应"
- 避免混杂偏差

## 技术细节

### Proxy ITE 的理论基础

**反事实干预法（Counterfactual Intervention）：**

对于每个样本 x：
1. 构造高暴露反事实样本：x_high = (x_genes, PM2.5=+1.0)
2. 构造低暴露反事实样本：x_low = (x_genes, PM2.5=-1.0)
3. Proxy ITE(x) = P(Y=1|x_high) - P(Y=1|x_low)

**核心原理：控制变量法（Ceteris Paribus）**
- 保持所有基因特征不变
- 仅改变 PM2.5 暴露水平
- 剥离基因背景的干扰
- 纯粹提取模型眼中的"PM2.5 效应"

**假设：**
- 分类模型的预测概率 P(Y=1|X) 近似于真实概率
- 模型能够捕捉 PM2.5 与基因的交互效应
- 反事实干预能够模拟真实的因果效应

**局限性：**
- Proxy ITE 不是真实 ITE，只是一个近似估计
- 依赖于模型的校准性（Calibration）和表达能力
- 不能用于精确的因果推断，仅用于机制验证
