# Deep-Lung-Causal (DLC) 项目结构说明

## 概述

本项目为 Deep-Lung-Causal (DLC) 因果推断模型，用于研究环境暴露（PM2.5）与肺癌风险之间的因果关系，特别关注基因-环境交互效应。
