# Deep-Lung-Causal (DLC) 项目结构说明

## 概述

本项目为 Deep-Lung-Causal (DLC) 因果推断模型，用于研究环境暴露（PM2.5）与肺癌风险之间的因果关系，特别关注基因-环境交互效应。

1. **XGBoost 最受益于数据增强**：能够有效利用跨癌种数据
2. **MOGONET 在小样本场景下表现最佳**：但在数据增强后性能下降

4. **HyperFast 需要进一步调试**：当前版本预测全部为正类，类别权重和阈值调整未生效

- TabR：分析检索池质量，尝试加权检索
