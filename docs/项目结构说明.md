# Deep-Lung-Causal (DLC) 项目结构说明

## 概述

本项目为 Deep-Lung-Causal (DLC) 因果推断模型，用于研究环境暴露（PM2.5）与肺癌风险之间的因果关系，特别关注基因-环境交互效应。

### 模块三: DLC 核心模型 (`src/dlc/`) ✅ NEW (Phase 6)

| 文件 | 功能 | 状态 |
|------|------|------|
| `causal_vae.py` | 因果变分自编码器 (CausalVAE) - 解耦混杂因素和效应因素 | ✅ 编码器已实现 (Task 1.2) |
| `hypergraph_nn.py` | 动态超图神经网络 (DynamicHypergraphNN) - 建模基因交互 | 🚧 开发中 |
| `dlc_net.py` | DLC 主模型 (DLCNet) - 端到端训练和预测 | 🚧 开发中 |
| `utils.py` | 工具函数 - 数据预处理、评估指标 | 🚧 开发中 |

**CausalVAE 实现进度**:
- ✅ `__init__`: 初始化编码器和解码器网络
- ✅ `encode`: 双编码器架构 (混杂编码器 + 效应编码器)
- ✅ `reparameterize`: 重参数化技巧实现
- ✅ `decode`: 从潜变量重构输入特征
- ✅ `forward`: 端到端前向传播
- 🚧 `compute_hsic_loss`: HSIC 独立性损失 (待实现)
- 🚧 `compute_recon_loss`: 重构损失 (待实现)

#### DLCNet 模块架构

```
DLCNet (主模型)
├── CausalVAE (因果解耦)
│   ├── Encoder_conf (混杂编码器) → Z_conf [B, 8]
│   ├── Encoder_effect (效应编码器) → Z_effect [B, 16]
│   └── Decoder (重构解码器) → X_recon [B, 23]
│
├── DynamicHypergraphNN (交互建模)
│   ├── Gene Embedding (基因嵌入) [B, 20, d_hidden]
│   ├── Multi-Head Attention (注意力机制) [B, 20, 20]
│   ├── Hypergraph Construction (超图构建) H [20, num_edges]
│   ├── Environment Gating (环境门控) H_gated
│   └── Hypergraph Convolution (超图卷积) → H_out [B, 20, d_hidden]
│
└── Outcome Heads (双头预测)
    ├── outcome_head_0 (Y(0) 预测) → [B, 1]
    └── outcome_head_1 (Y(1) 预测) → [B, 1]
```

#### 关键功能

1. **前向传播** (`forward`)
   - 输入：X [B, 23] (Age, Gender, PM2.5, Top 20 Genes)
   - 输出：
     - Y_0: 未暴露结局预测 [B, 1]
     - Y_1: 暴露结局预测 [B, 1]
     - ITE: 个体治疗效应 [B, 1]
     - Z_conf: 混杂表征 [B, 8]
     - Z_effect: 效应表征 [B, 16]
     - X_recon: 重构特征 [B, 23]

2. **损失计算** (`compute_loss`)
   - L_total = L_recon + λ_hsic * L_hsic + λ_pred * L_pred
   - L_recon: 重构损失（MSE + BCE）
   - L_hsic: HSIC 独立性损失（确保 Z_conf ⊥ Z_effect）
   - L_pred: 预测损失（Binary Cross-Entropy）

3. **三阶段训练** (`fit`)
   - 阶段 1: VAE Warmup (20 epochs)
     - 仅训练 CausalVAE
     - 优化 L_recon + λ_hsic * L_hsic
     - 学习率：1e-3
   - 阶段 2: Interaction Learning (30 epochs)
     - 冻结 VAE，训练超图和预测头
     - 优化 λ_pred * L_pred
     - 学习率：5e-4
   - 阶段 3: Joint Fine-tuning (50 epochs)
     - 联合微调所有模块
     - 优化完整损失 L_total
     - 学习率：1e-4（带学习率调度）
     - Early Stopping（patience=10）

4. **预测接口**
   - `predict_proba(X)`: 返回 [N, 2] 概率矩阵
   - `predict(X)`: 返回 {0, 1} 标签
   - `evaluate(X, y)`: 返回 accuracy, precision, recall, f1, auc_roc

#### 模块详细说明

##### 1. CausalVAE (因果变分自编码器)

**核心功能**: 解耦混杂因素和效应因素

**架构设计**:
```
输入 X [B, 23] (Age, Gender, PM2.5, Top20 Genes)
    ↓
┌─────────────────────────────────────┐
│  双编码器架构                        │
│  ├─ encoder_conf → (μ_conf, σ_conf) │  [B, 8]
│  └─ encoder_effect → (μ_effect, σ_effect) │  [B, 16]
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  重参数化采样                        │
│  ├─ Z_conf = μ_conf + σ_conf * ε    │
│  └─ Z_effect = μ_effect + σ_effect * ε │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  解码器                              │
│  decoder([Z_conf; Z_effect]) → X_recon │  [B, 23]
└─────────────────────────────────────┘
```

**损失函数**:
- `L_recon`: 重构损失（MSE + BCE）
- `L_hsic`: HSIC 独立性损失（确保 Z_conf ⊥ Z_effect）

**关键参数**:
- d_conf = 8: 混杂因素维度
- d_effect = 16: 效应因素维度
- hidden_dim = 64: 隐藏层维度

**测试结果**:
- 10 个单元测试全部通过
- HSIC 独立性验证通过（独立变量 HSIC < 0.1）
- 重构质量验证通过（MSE < 0.1）



##### 2. DynamicHypergraphNN (动态超图神经网络)

**核心功能**: 建模基因-环境交互

**架构设计**:
```
输入: X_gene [B, 20], Z_effect [B, 16]
    ↓
┌─────────────────────────────────────┐
│  基因嵌入层                          │
│  gene_embedding(X_gene) → X_emb     │  [B, 20, d_hidden]
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  多头自注意力                        │
│  MultiheadAttention(X_emb) → attn   │  [B, 20, 20]
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  超图构建（Top-K 选择）              │
│  TopK(attn, k=5) → H                │  [B, num_edges, 20]
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  环境门控（核心创新）                │
│  gate = sigmoid(W @ Z_effect)       │
│  H_gated = gate * H                 │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  超图卷积（多层堆叠）                │
│  HypergraphConv(X_emb, H_gated)     │  [B, 20, d_hidden]
└─────────────────────────────────────┘
```

**关键参数**:
- num_genes = 20: 基因数量
- d_effect = 16: 效应表征维度
- d_hidden = 64: 隐藏层维度
- num_heads = 4: 注意力头数
- num_layers = 2: 超图卷积层数
- num_edges = 80: 超边数量
- k = 5: Top-K 选择参数

**测试结果**:
- 8 个单元测试全部通过
- 超边度数约束验证通过（每个超边至少连接 2 个节点）
- 节点度数约束验证通过（每个节点至少属于 1 个超边）
- 门控系数范围验证通过（[0, 1]，无 NaN/Inf）



##### 3. DLCNet (主模型)

**核心功能**: 端到端因果推断和结局预测

**架构设计**:
```
输入 X [B, 23]
    ↓
┌─────────────────────────────────────┐
│  特征分割                            │
│  ├─ X_clinical = X[:, :3]           │  (Age, Gender, PM2.5)
│  └─ X_gene = X[:, 3:]               │  (Top20 Genes)
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  CausalVAE (因果解耦)                │
│  ├─ Z_conf [B, 8]                   │
│  ├─ Z_effect [B, 16]                │
│  └─ X_recon [B, 23]                 │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  DynamicHypergraphNN (交互建模)      │
│  H_out = hypergraph_nn(X_gene, Z_effect) │  [B, 20, 64]
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  全局池化                            │
│  H_global = mean(H_out, dim=1)      │  [B, 64]
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  双头预测器                          │
│  ├─ outcome_head_0 → Y(0) [B, 1]    │  (未暴露结局)
│  └─ outcome_head_1 → Y(1) [B, 1]    │  (暴露结局)
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  ITE 计算                            │
│  ITE = Y(1) - Y(0)                  │  [B, 1]
└─────────────────────────────────────┘
```

**损失函数**:
```
L_total = L_recon + λ_hsic * L_hsic + λ_pred * L_pred

其中:
- L_recon: 重构损失（MSE + BCE）
- L_hsic: HSIC 独立性损失
- L_pred: 预测损失（Binary Cross-Entropy）
- λ_hsic = 0.1
- λ_pred = 1.0
```

**三阶段训练策略**:
1. **阶段 1: VAE Warmup** (20 epochs)
   - 仅训练 CausalVAE
   - 优化 L_recon + λ_hsic * L_hsic
   - 学习率: 1e-3
   - 目标: HSIC < 0.01

2. **阶段 2: Interaction Learning** (30 epochs)
   - 冻结 VAE，训练超图和预测头
   - 优化 λ_pred * L_pred
   - 学习率: 5e-4
   - 目标: Val AUC > 0.70

3. **阶段 3: Joint Fine-tuning** (50 epochs)
   - 联合微调所有模块
   - 优化完整损失 L_total
   - 学习率: 1e-4（带 ReduceLROnPlateau 调度器）
   - Early Stopping（patience=10）
   - 目标: Val AUC > 0.80

**测试结果**:
- 13 个单元测试全部通过
- 端到端训练测试通过（100 样本，2 epochs）
- 端到端工作流测试通过（200 训练 + 50 测试，2 epochs）

#### Task 7.1: 高级因果指标模块

**目标**: 创建 `src/dlc/metrics.py`，实现三个核心因果推断评估函数

**功能需求**:
1. `compute_pehe(y_true_ite, y_pred_ite) -> float`
   - 计算 PEHE (Precision in Estimation of Heterogeneous Effect)
   - 公式：`sqrt(mean((y_true_ite - y_pred_ite)^2))`
   - 支持 NumPy 数组和 PyTorch Tensor
   - 处理 NaN 值

2. `compute_cate(model, X, treatment_col_idx) -> np.ndarray`
   - 计算 CATE (Conditional Average Treatment Effect)
   - 方法：反事实干预法 (Counterfactual Intervention)
   - 支持批量计算

3. `compute_sensitivity_score(model, X, confounder_idx, epsilon=0.1) -> float`
   - 计算混杂敏感度分数
   - 支持多个混杂变量
   - 归一化到 [0, 1] 范围

**测试需求**:
- 单元测试：`tests/test_metrics.py`（4 个测试用例）
- 属性测试：`tests/test_metrics_properties.py`（3 个属性）
