### 2025-01-15 Task 7.3 修复：PM2.5 混杂效应属性测试

**操作者：** Kiro

**问题分析：**

原始 PM2.5 生成公式：`Virtual_PM2.5 = 30 + 0.5 * Z(Age) + N(0, 10)`

理论相关系数计算：
- Age 系数 = 0.5，噪声标准差 = 10
- 理论相关系数 ≈ 0.5 / √(0.25 + 100) ≈ **0.05**

这是一个非常弱的相关性，在有限样本下很容易因采样误差而出现负值。

**用户决策：** 选择【更新代码实现】，将 Age 系数从 0.5 提高到 3.0 以增强混杂信号。

**修复内容：**

1. **更新 `src/data_processor.py`**
   - 新增类常量：`PM25_BASE = 30.0`, `PM25_AGE_COEF = 3.0`, `PM25_NOISE_STD = 10.0`
   - 修改 `generate_pm25()` 方法使用新公式：`30 + 3.0 * Z(Age) + N(0, 10)`
   - 新理论相关系数 ≈ 3.0 / √(9 + 100) ≈ **0.29**

2. **更新需求文档 `.kiro/specs/dlc-data-engineering/requirements.md`**
   - 将公式描述从 `0.5 * Z(Age)` 改为 `3.0 * Z(Age)`

3. **更新设计文档 `.kiro/specs/dlc-data-engineering/design.md`**
   - 更新 `generate_pm25()` 方法的公式注释

4. **更新测试断言 `tests/test_data_processor_properties.py`**
   - 将相关系数断言从 `> 0` 改为 `> 0.1`（留有余量）

   - 计算 5 个评估指标：
     - accuracy: 准确率
     - precision: 精确率（二分类）
     - recall: 召回率（二分类）
     - f1: F1 分数（二分类）
     - auc_roc: ROC 曲线下面积

# 内部验证集划分（15%）
X_train, X_val, y_train, y_val = train_test_split(
    X, y, 
    test_size=0.15, 
    random_state=self.random_state, 
    stratify=y  # 保持类别比例
)

# 初始化 XGBoost 分类器
self.model = xgb.XGBClassifier(
    n_estimators=self.n_estimators,
    learning_rate=self.learning_rate,
    max_depth=self.max_depth,
    min_child_weight=self.min_child_weight,
    subsample=self.subsample,
    colsample_bytree=self.colsample_bytree,
    random_state=self.random_state,
    eval_metric='logloss',
    early_stopping_rounds=10
)

# 使用内部验证集进行 Early Stopping
self.model.fit(
    X_train, y_train,
    eval_set=[(X_val, y_val)],
    verbose=False
)

### Proxy ITE 计算逻辑修正（2026-01-23 更新）

**问题发现：**
原始的简化估计存在严重的混杂偏差（Confounding Bias）：
```python
# 错误方法
proxy_ite = y_proba - mean_proba_t0
```

**修正方案：反事实干预法**

采用控制变量法（Ceteris Paribus）剥离基因背景干扰：

```python
def compute_proxy_ite(model, X, pm25_feature_idx):
    # 构造高暴露样本（PM2.5 = +1.0）
    X_high = X.copy()
    X_high[:, pm25_feature_idx] = 1.0
    
    # 构造低暴露样本（PM2.5 = -1.0）
    X_low = X.copy()
    X_low[:, pm25_feature_idx] = -1.0
    
    # 计算 Proxy ITE
    proxy_ite = model.predict_proba(X_high) - model.predict_proba(X_low)
    
    return proxy_ite
```

**核心原理：**
- 保持所有基因特征不变
- 仅改变 PM2.5 暴露水平
- 纯粹提取模型眼中的"PM2.5 效应"
- 避免混杂偏差
